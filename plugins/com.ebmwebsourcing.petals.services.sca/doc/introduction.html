Introduction

About this documentation

First, let's be clear about the fact that this documentation does aim at introducing SCA in a general way. It is assumed you are already familiar with the SCA specification. If not, you should check out the links given at the bottom of this page.

This documentation explains the requirements for SCA applications to run into Petals. It also introduces the SCA tools which were created specifically for Petals and how they are related to the SCA tools of the Eclipse STP project.
